DOM的理解

DOM指的是 Document Object Model(文档对象模型), 它是JavaScript和网页的桥梁, 能够使JavaScript操作网页中的元素。

DOM将页面中所有的内容都抽象为一个个可以修改的对象，这个抽象过程称之为文档对象模型。

比如说：

  1. 整个文档-被抽象到 document 对象中
  2. html元素-被抽象到 document.documentElement
  3. head元素-被抽象到 document.head

DOM Tree的理解

在页面结构中，最终会形成一个 树结构

当 html tree 抽象成DOM对象时，就形成了DOM Tree

DOM的继承关系图

document对象

Document节点表示的整个载入的网页，它的实例是全局的document对象

对DOM的操作都是从document对象开始的

它是DOM的 入口点，可以从document开始去访问任何节点元素

节点(Node)之间的导航(navigator)

获取到一个节点 Node 之后，可以根据这个节点获取其他的节点，这就称为节点之间的导航。

元素(Element)之间的导航(navigator)

当我们知道Node之间的导航获取的是全部的节点后, 我们发现获取到很多的换行符, 注释等一些在开发中并不需要获取的节点

在开发中我们更关心的是获取 元素, 所以这就需要用到元素之间的导航

节点Node常见属性

nodeType属性: 返回节点类型的数值

nodeName属性: 返回节点的名称

tagName属性: 只返回元素的名称

data/nodeValue属性: 针对非元素的节点获取文本内容

innerHTML: 将元素中HTML获取为字符串形式

outerHTML: 包含了元素完整的HTML

textContent: 仅仅获取元素中的文本内容

Element元素的Attribute

attribute的分类

  1. 标准的attribute: 如id, class, href, type, value等
  2. 非标准的attribute: 自定义的attribute属性, 如abc, age等

attribute的操作

对于所有的attribute都支持如下操作:

el.hasAttribute(name) - 检查attribute是否存在

el.getAttribute(name) - 获取这个attribute的值

el.setAttribute(name, value) - 设置这个特性值

el.removeAttribute(name) - 移除这个特性

attributes: attr对象的集合, 具有name和value属性

attribute的特性:

  1. 名字是大小写不敏感的(class和CLASS相同)
  2. 它们的值总是字符串类型

Element元素的Property

对象中的属性称之为property, 元素中的属性称之为attribute

标准的Attribute在对应的对象模型中都有对应的property

非标准的Attribute只能通过attribute操作获取对应的property

Element元素的Class/Style

JavaScript动态修改Class

ClassName

会将给定的class类名覆盖掉原来的class类名

classList对象

  1. el.classList.add(token1[, token2[, …tokenN]]) : 添加类
  2. el.classList.remove((token1[, token2[, …]])): 移除类
  3. el.classList.replace(oldToken, newToken):已存在的 token 替换为一个新 token, 如果第一个参数 token 在列表中不存在， replace() 立刻返回false ，而不会将新 token 字符串添加到列表中。
  4. el.classList.toggle(class):如果类不存在就添加类, 存在就移除它
  5. el.classList.contains(class):检查给定类, 返回true/false
