学习JavaScript的Date对象

AHAI
2023-09-08
JavaScript, 学习笔记

创建日期对象

1. 不带参数

显示当前日期和时间

2. 时间戳

表示1970年1月1日UTC+0之后经过的毫秒数

3. 字符串 参数

4. new Date()

new Date(year, month, date, hours, minutes, seconds, ms)

  • year 和 month 是必须的两个参数，其余参数不传则默认为 0
  • year 强烈建议使用 4 位数 ：2002
  • month 从 0（一月）到 11 （十二月）*

访问日期时间

1.getFullYear()

返回4位数年份,区别于非标准化的方法 getYear() 有可能返回的2位数年份

2.getMonth()

获取月份从 0 - 11

3.getDate()

获取当月具体日期 1-31

4.getHours()，getMinutes()，getSeconds()，getMilliseconds()获取相应的时间组件。

5.getDay()

获取一周中的第几天 0(星期天)-6(星期6)

6.getTime()

将现有的Date对象转化为时间戳

7.getTimezoneOffset()

返回UTC与本地时区之间的时差，以分钟为单位

8.Date.now()

Date类的静态方法,不需要new一个Date对象,因此它更快,不会对垃圾回收造成额外的压力
返回当前的时间戳, 相当于new Date().getTime()
此方法因方便和性能优越常常被考虑

9.valueOf()

valueOf = getTime() = now() = +new Date()

设置日期时间

下列方法可以设置日期/时间组件：
1.setFullYear(year, [month], [date])
2.setMonth(month, [date]) 返回设置后的时间戳
3.setDate(date)
4.setHours(hour, [min], [sec], [ms]) 返回设置后的时间戳
5.setMinutes(min, [sec], [ms])
6.setSeconds(sec, [ms])
7.setMilliseconds(ms)
8.setTime(milliseconds)（使用自 1970-01-01 00:00:00 UTC+0 以来的毫秒数来设置整个日期）

以上方法除了 setTime() 都有 UTC 变体，例如：setUTCHours()。

自动校准

1.超出的范围的日期数值,会自动校准
2.计算日期, Date对象会自动帮处理: 如 2月28 + 2 日, 我们需要去判断是否闰年,而计算机直接自动帮助判断
3.此特性通常会被用来获取给的那个时间段后的日期: 如 现在 100秒 后的日期

日期转换为数字

Date对象被转化为数字时,得到的是对应的时间戳

日期差值

日期可以相减, 相减的结果是以毫秒为单位时间差

对日期字符串调用Date.parse

Date.parse(str)方法可以从一个字符串中读取日期
会解析给定字符串,返回时间戳. 字符串格式不正确, 返回NaN
字符串格式应该为: YYYY-MM-DDTHH:mm:ss.sssZ
1.YYYY-MM-DD —— 日期：年-月-日。
2.字符 “T” 是一个分隔符。
3.HH:mm:ss.sss —— 时间：小时，分钟，秒，毫秒。
4.可选字符 ‘Z’ 为 +-hh:mm 格式的时区。单个字符 Z 代表 UTC+0 时区。
5.简短形式也是可以的，比如 YYYY-MM-DD 或 YYYY-MM，甚至可以是 YYYY。

关于高精度时间

JavaScript中时间戳以毫秒为单位，而不是秒。

当我们需要更精准的时间度量时，JavaScript自身并没有测量微妙的方法。

但是大多数运行环境会提供：

  1. 浏览器：==performance.now() ==：从页面加载开始的以毫秒为单位的微妙数（精确到毫秒的小数点后三位）
  2. Node.js: ==microtime==

高级的日期格式化

  1. 使用第三方库moment.js 日期和时间库
  2. 使用通用性较高的格式化函数

这个函数接受两个参数，第一个参数是要格式化的日期时间，可以是 Date 对象或表示日期时间的字符串，第二个参数是要格式化的格式，例如 ==yyyy-MM-dd HH:mm:ss==。该函数会将日期时间格式化为指定的格式，并返回格式化后的字符串。

该函数使用了正则表达式来匹配格式字符串中的占位符，然后根据对应的日期时间值来替换占位符。其中，y 会被替换为年份，M、d、h、H、m、s、q、S、a、A 分别表示月份、日期、小时（12 小时制）、小时（24 小时制）、分钟、秒、季度、毫秒、上午/下午、AM/PM。
