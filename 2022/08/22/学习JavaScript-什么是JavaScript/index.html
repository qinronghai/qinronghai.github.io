学习JavaScript-什么是JavaScript

AHAI
2022-08-22
JavaScript, 学习笔记

什么是JavaScript

历史

网景工程师为Netscape Navigator 2开发一个脚本语言：Mocha（后来改名为 LiveScript），在Netscape Navigator 2 正式发布前，网景把 LiveScript 改名为 JavaScript，以便搭上媒体当时热烈炒作 Java 的顺风车。后来经过当时很多大厂的对这门脚本语言有兴趣的工程师打造出了 ECMA-262，也就是 ECMAScript（发音为“ek-ma-script”）这个新的脚本语言标准。

背景

在JavaScript问世之前，要验证某个必填字段是否已填写，或值是否有效，需要与服务器进行一次往返通信。加之当时网速很慢，往返通信的时间很长，所以能够使用JavaScript在客户端就能处理这些基本的验证是让人兴奋的。

备注

  • 脚本语言与非脚本语言

    脚本语言就像解释型语言，翻译一行就执行一行，无需关注往后的代码 ，哪怕后面有些内容是错误的，也不会影响前面语句的执行。

    非脚本语言就是一定要全部的代码没有错误，然后编译通过之后才可以运行起来。

总结

​ JavaScript 是一门用来与网页交互的脚本语言，包含以下三个组成部分。

  • ECMAScript：由 ECMA-262 定义并提供核心功能。
  • 文档对象模型（DOM）：提供与网页内容交互的方法和接口。
  • 浏览器对象模型（BOM）：提供与浏览器交互的方法和接口。
