seaEliy

Linux, 操作系统

作业

实验安装、运行Linux、使用开发环境
实验内容：

  1. 安装虚拟机：virtualbox或VMware
  2. 在虚拟机中安装Linux：centos或ubuntu
  3. 在Linux，使用编辑器偏辑C语言HelloWorld源代码程序：vim或图形编辑器
  4. 在Linux，使用gcc命令编译C程序，运行程序：如果没有gcc，先安装gcc

下载Ubuntu镜像文件

下载地址：https://ubuntu.com/

  1. 点击上面链接，点击Download

  2. 点击Ubuntu Desktop

  3. 点击这个

  4. 我下的是这个

  5. 下载桌面版，下载好之后新建文件夹放好，后面要用到。

VMware 下载安装

下载地址：https://www.vmware.com/cn/products/workstation-pro/workstation-pro-evaluation.html

  1. 下载安装

  2. 有个输入许可证密钥按钮

  3. 打开VMware pro，创建新的虚拟机

  4. 选择自定义

  5. 下一步再下一步，到这选择稍后安装操作系统

  6. 选择Linux，版本下来选择Ubuntu64

  7. 选择安装位置，这里必须输入一个已存在的目录，否则后面会报错

  8. 配置选择默认，下一步

  9. 默认，下一步

  10. 默认

  11. 一直下一步到这

  12. 下一步到这

  13. 选择第一步下载的Ubuntu镜像（即前面下载好的Ubuntu）

  14. 点击完成，虚拟机到此配置完毕，开启虚拟机

  15. 等加载

  16. 到这里选择中文，点击安装

  17. 双击选择的键盘布局chinese

  18. 到这默认

  19. 点击现在安装，一直继续，等

  20. 选择上海

  21. 设置用户名和密码

  22. 正式安装，等待一段时间

  23. 安装完成（可以打开浏览器检查一下网络是否可用）

Ubuntu下编写并运行第一个c语言程序

使用快捷键Ctrl+Alt+T打开控制台;

安装vim

在控制台输入 sudo apt-get install vim,遇到输入密码时回车

安装gcc

在控制台输入sudo apt-get install g++

编写hello.c源代码

  1. 同样在控制台，输入 vim hello.c

  3. 输入完成后，按 Esc 键回到normal模式，随后键入:wq 保存退出vim。

  4. 编译hello.c,在终端执行 g++ hello.c -o hello 编译。

  5. 运行程序hello，键入 ./hello 就可以看到结果。

至此，完成。
